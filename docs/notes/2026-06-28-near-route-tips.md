# 邻近区间提示统一（2026-06-28）

## 问题
- 安房↔宫之浦等「港/镇」近似区间，此前 `DEST/ORIGIN_STOP_ALTS` 不全（如缺 `anbo→anbo_port`、`miyanoura` 到达备选），有班次时不提示
- 文案按总站数对比易误解（如「16 班」实为港站总数，非多出 16 班）

## 方案
- `STOP_NEAR_ALTS` 统一港/镇/入口矩阵；从港出发不再推「改镇上」（保留 `PORT_FROM_STOPS`）
- 枚举邻近起终点组合，按 `route.id|arr` 去重后只提示**多出来的班次**，最多 3 条
- `#routeTips` 有结果时也显示；每条附发车时刻 chips + 一键改搜
- 例：安房→宫之浦（10 班）会列出安房港→宫之浦港（+11）、安房→宫之浦港（+5）等
