# 徒步路线卡片 UX 重排（2026-05-20）

## 用户场景优先级
1. 这是什么线、我能不能走（标题 + 简介 + 用时/距离/爬升）
2. 难不难（难度/体能一行，非表格）
3. 怎么去（主 CTA「查接驳」→ 预设区间时刻表）
4. 深读（观光协会 / YAMAP 次要链接）

## 改动
- 去掉 2×2 标签表格、多层 border、每卡免责声明
- 核心数据改为 chips；公交区单一主按钮 + 区间 + 一句提示
- 区块级一句「以官方时刻表为准」

## 2026-05-20 更新
