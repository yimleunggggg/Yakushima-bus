# 首页中文 i18n 修复（2026-06-25）

## 现象
`/?lang=zh` 时 H1 仍显示日文「バス時刻表」；松ばんだ运营商显示假名；冬季班次备注可能出现「悠享」字体乱码。

## 根因
1. `index.html` `applyPageHead()` 未传 `lang`，且 SiteChrome 未加载时无 fallback 标题。
2. `site-chrome-nav-boot.js` 的 `syncPageTitle()` 未覆盖 `/` 首页。
3. `site-chrome.js` 在 body 末尾加载，晚于首屏 inline 脚本。
4. `data.js` matsubanda 中文仍为假名「松ばんだ」。

## 修复
- `site-chrome-nav-boot.js`：启动时同步 `document.documentElement.lang`；`syncPageTitle` 覆盖 `/` 及 ferry/trekking/intro。
- `index.html`：传 `lang`、fallback 标题、`site-chrome.js` 移至 `<head>`；缓存 `chrome-v16`。
- `data.js` / `operators.py`：matsubanda zh →「松叶交通」/「松叶」；冬季备注去掉产品名。

## 预览
`http://localhost:8765/?lang=zh`
