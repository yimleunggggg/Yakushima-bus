## 二、其他页面怎么「编入索引」

只有首页 PASS 不够；**/map/、/access/、/about/** 需要同样做一遍（各 **一次** 即可，不用每天点）。

### 操作步骤（GSC 左上角「网址检查」）

1. 输入完整 URL，例如 `https://yakushimabus.com/map/`
2. 等检测完成
3. 若显示 **「URL 不在 Google 上」** 或 **未编入索引** → 点 **「请求编入索引」**
4. 对下面三个 URL **各重复一次**：
   - `https://yakushimabus.com/map/`
   - `https://yakushimabus.com/access/`
   - `https://yakushimabus.com/about/`
5. 已显示 **Page is indexed** 的不用重复请求

### 还需要确认的（一次性）

- [ ] GSC → **站点地图** → 已提交 `https://yakushimabus.com/sitemap.xml`
- [ ] GSC → **编制索引 → 网页** → 看 4 页是否都从 NEUTRAL → 已编入索引

### 索引之后 SEO 还要做什么

索引 ≠ 有流量。页面进索引后，Google 才会在「屋久島 バス 運賃」等词上试推 **/map/**，在船票词上试推 **/access/**。
