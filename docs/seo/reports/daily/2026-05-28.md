| 美国/欧洲用户但互动 0s | 多为爬虫/VPN（已剔除：Canada(3), Poland(3), South Korea(3)…）。自测加 ?ga_internal=1。 |
| GA4 总用户 51 vs 可分析 39 | 总用户含 bot；可分析=去掉互动0%+均时0s的国家行。 |
| 洞察 | 已排除 **疑似 bot** 12 用户 / 12 会话（Canada(3), Poland(3), South Korea(3), Austria(1), France(1), India(1)）：互动 0% 且均时 0s。**可分析**：约 39 用户 / 54 会话。 |
| 洞察 | **Direct 92% + Unassigned 2%** 占主导（渠道为 GA4 原始值，bot 多记 Direct）：朋友圈/书签等 **无 referrer** 是主因，不是 GA4 漏记。 |
| 洞察 | 来源明细 Top：`(direct), (not set), github.com`（见 §2.2）。 |
