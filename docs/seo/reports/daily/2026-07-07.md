### 2.0 这些指标是什么意思

| 指标 | 怎么读 |
|---|---|
| **活跃用户** | 去重访客（靠 `_ga` Cookie，不是 IP） |
| **会话** | 一次访问；同一人一天内可能多次 |
| **互动率 / 均时** | 是否「真在看」：≥10s 或多页 ≈ 参与；0s 常是 bot 或误点 |
| **Direct** | 无 referrer，**含微信**；不是「直接输入网址」 |
| **Organic** | 从 Google/Bing 等搜索引擎点进来 |
| **GSC 展示** | 链接出现在搜索结果里；**≠** 点进网站 |
| **索引 PASS/NEUTRAL** | 收录状态；NEUTRAL 可手动请求编入索引 |

| 美国/欧洲用户但互动 0s | 多为爬虫/VPN（已剔除：Australia(1), Italy(1), Singapore(1)…）。自测加 ?ga_internal=1。 |
| GA4 总用户 127 vs 可分析 124 | 总用户含 bot；可分析=去掉互动0%+均时0s的国家行。 |
| 判断「有人在用」 | 多着陆页+互动率不低，在查时刻表/地图。 |
| 洞察 | 已排除 **疑似 bot** 3 用户 / 3 会话（Australia(1), Italy(1), Singapore(1)）：互动 0% 且均时 0s。**可分析**：约 124 用户 / 150 会话。 |
| 洞察 | 自然搜索占 **65%**（100 会话）：SEO 开始起量，与 GSC 点击交叉验证。 |
| 洞察 | 欧盟/英国 **1** 用户、互动率 **100.0%**：有实际浏览行为，可能含欧州赴日旅客或英文搜索；仍建议对照着陆页是否多页访问。 |
