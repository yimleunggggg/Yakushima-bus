## 三、为什么会这样

| 现象 | 说明 |
| --- | --- |
| 渠道几乎全是 Direct | 微信/朋友圈/书签不带 referrer，GA4 记成 Direct；看国家+均时判断真人，不是统计坏了。 |
| 美国/欧洲用户但互动 0s | 多为爬虫/VPN（已剔除：South Korea(2), Liechtenstein(1)…）。自测加 ?ga_internal=1。 |
| GA4 总用户 45 vs 可分析 42 | 总用户含 bot；可分析=去掉互动0%+均时0s的国家行。 |
| 洞察 | 已排除 **疑似 bot** 3 用户 / 3 会话（South Korea(2), Liechtenstein(1)）：互动 0% 且均时 0s。**可分析**：约 42 用户 / 57 会话。 |
| 洞察 | **Direct 77% + Unassigned 5%** 占主导（渠道为 GA4 原始值，bot 多记 Direct）：朋友圈/书签等 **无 referrer** 是主因，不是 GA4 漏记。 |
| 洞察 | 来源明细 Top：`(direct), (not set), chatgpt.com`（见 §2.2）。 |
| 洞察 | 欧盟/英国 **2** 用户、互动率 **50.0%**：有实际浏览行为，可能含欧州赴日旅客或英文搜索；仍建议对照着陆页是否多页访问。 |
