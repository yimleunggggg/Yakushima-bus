## 学一点

- **Organic Search 渠道**：GA4 里自然搜索会话上涨，说明 SEO 起量；与 GSC 点击交叉验证。Direct 多可能是书签、微信分享链接直达。
- **索引 PASS / NEUTRAL**：PASS 表示 Google 已收录可用；NEUTRAL 常是新页或次要页，可在 Search Console 对该 URL「请求编入索引」（需手动，无稳定全自动 API）。
