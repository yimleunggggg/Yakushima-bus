### 2.0 这些指标是什么意思

| 指标 | 怎么读 |
|---|---|
| **活跃用户** | 去重访客（靠 `_ga` Cookie，不是 IP） |
| **会话** | 一次访问；同一人一天内可能多次 |
| **互动率 / 均时** | 是否「真在看」：≥10s 或多页 ≈ 参与；0s 常是 bot 或误点 |

| **Organic** | 从 Google/Bing 等搜索引擎点进来 |
| **GSC 展示** | 链接出现在搜索结果里；**≠** 点进网站 |
| **索引 PASS/NEUTRAL** | 收录状态；NEUTRAL 可手动请求编入索引 |

| GA4 总用户 101 vs 可分析 96 | 总用户含 bot；可分析=去掉互动0%+均时0s的国家行。 |
| 判断「有人在用」 | 多着陆页+互动率不低，在查时刻表/地图。 |
| 洞察 | 已排除 **疑似 bot** 5 用户 / 5 会话（Hong Kong(1), Singapore(1), Türkiye(1), Ukraine(1), Vietnam(1)）：互动 0% 且均时 0s。**可分析**：约 96 用户 / 128 会话。 |
| 洞察 | 自然搜索占 **49%**（65 会话）：SEO 开始起量，与 GSC 点击交叉验证。 |
| 洞察 | 近 7 日国家 Top：**Japan(63), United States(22), China(11), South Korea(1)**（无欧盟/英国条目或量极低）。 |
| 洞察 | 多着陆页 + 7 日互动率 **42.9%**：有人在用时刻表/地图等工具，非纯误点。 |
