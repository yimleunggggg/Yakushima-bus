### 2.0 这些指标是什么意思

| 指标 | 怎么读 |

| **活跃用户** | 去重访客（靠 `_ga` Cookie，不是 IP） |

## 三、为什么会这样

| 现象 | 说明 |
| --- | --- |
| 渠道几乎全是 Direct | 微信/朋友圈/书签不带 referrer，GA4 记成 Direct；看国家+均时判断真人，不是统计坏了。 |
| 美国/欧洲用户但互动 0s | 多为爬虫/VPN（已剔除：Canada(2), South Korea(2), Austria(1)…）。自测加 ?ga_internal=1。 |
| GA4 总用户 38 vs 可分析 32 | 总用户含 bot；可分析=去掉互动0%+均时0s的国家行。 |
| ⚠ 告警 | 若你本人用欧洲节点 VPN 测站且未加 `?ga_internal=1`，会计入此类。 |
| 洞察 | 已排除 **疑似 bot** 6 用户 / 6 会话（Canada(2), South Korea(2), Austria(1), India(1)）：互动 0% 且均时 0s。**可分析**：约 32 用户 / 41 会话。 |
| 洞察 | **Direct 85% + Unassigned 13%** 占主导（渠道为 GA4 原始值，bot 多记 Direct）：朋友圈/书签等 **无 referrer** 是主因，不是 GA4 漏记。 |
| 洞察 | 来源明细 Top：`(direct), (not set), github.com`（见 §2.2）。 |
| 洞察 | 欧盟/英国用户 **1**（Estonia(1)），互动率均值 **0.0%**、会话≈用户：更像 **单次跳出/爬虫/VPN**，暂不能当真实旅客。 |
